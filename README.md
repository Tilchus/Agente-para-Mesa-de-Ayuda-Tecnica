## 🚀 Guía de Configuración Inicial

Para poner en marcha el agente, seguí estos pasos al pie de la letra:

### 🔑 1. Obtención de la API KEY
Para que la inteligencia del agente funcione, necesitás una clave de Google:
1. Registrate en [Google AI Studio](https://aistudio.google.com/).
2. En el menú lateral izquierdo, hacé clic en el ícono de **llave (Get API key)**.
3. Hacé clic en **"Create API key"**. 
4. Copiá la clave generada y guardala en un lugar seguro.

### 🛡️ 2. Configuración del archivo de seguridad (.env)
*Es vital que no escribas tu clave directamente en el código para mantener la seguridad.*
1. Creá un archivo nuevo en la carpeta del proyecto.
2. Al guardar (**Archivo > Guardar como...**), en "Tipo" seleccioná **"Todos los archivos (*.*)"**.
3. Poné de nombre exactamente: `.env`
4. Pegá tu clave así: `GOOGLE_API_KEY=PEGÁ_ACÁ_TU_CLAVE`
5. Verificá que el archivo se llame `.env` y no `.env.txt`.

### 📦 3. Instalación de librerías
Desde la terminal, ejecutá el comando para descargar las dependencias:
`pip install -r requirements.txt`

### ▶️ 4. Ejecución del Agente
Una vez configurado, ejecutá el siguiente comando en tu terminal:
`python "Agente de ayuda Tecnica.py"`
